# Vibe-Tune 🎵 – Intelligent Mood Music Player (Android)

Vibe-Tune is an Android application that plays music based on the user's selected mood. The app integrates Spotify links to open songs or playlists corresponding to different moods such as Happy, Sad, Calm, Energetic, Romantic, etc.

## 📱 Project Overview
The application allows users to select their mood from a dropdown list. Based on the selected mood, the app displays a mood image and opens a Spotify track or playlist.

If the Spotify app is installed, the track opens in Spotify. Otherwise, it opens in a web browser.

## 🚀 Features
- Mood-based music selection
- Spotify integration
- Playlist support
- Mood images
- Simple UI using Spinner
- Opens Spotify app or browser fallback
- Supports multiple moods and languages

## ▶️ How to Run the App
1. Open project in Android Studio
2. Connect Android device or emulator
3. Click Run
4. Select mood
5. Tap Play
6. Music opens in Spotify or browser
